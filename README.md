
# 🎬 Netflix Data Analysis

## 📌 Overview
This project analyzes Netflix content using Python to extract insights about movies, TV shows, and production trends.

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib

## 📊 Key Insights
- Distribution of Movies vs TV Shows
- Top content-producing countries
- Trends in Netflix content

## ▶️ How to Run
```bash
pip install pandas matplotlib
python analysis.py
